# 枢元 (ShuYuan AI)

**智能中枢 (枢) + 元认知进化 (元) 的自适应治理系统**

> *枢以定势，元以驭变* - 受古代治理智慧启发的下一代 AI 协作系统

> *致敬开源*：本项目受 [Edict](https://github.com/cft0808/edict) 项目启发，在其"三省六部制"AI多Agent协作架构基础上，扩展了动态路由、自我进化、价值密度优化等核心能力。

**​"枢"​** 代表 **枢机院**——智能路由中枢，负责情境感知、任务分类，将指令动态路由到快反轨道、常规轨道或圆桌会议轨道，是系统"动态适应性"的关键体现。

**​"元"​** 代表 **元认知与自我进化**——系统的最高阶能力，指向"自我进化引擎"、元制度设计，以及系统能够反思、优化、甚至重新定义自身治理方式的能力，触及"治理的治理"这一元层面。

**​"枢元"​** 二字结合，精准概括了系统的精髓：**一个以智能中枢(枢)为调度核心，以元认知进化(元)为驱动力的自适应治理系统**。
